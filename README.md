# 카커넥트 차량 카탈로그

GitHub Pages에 올려 고객에게 공유할 수 있는 차량 검색형 홈페이지입니다.

## 업로드 방법

1. GitHub 로그인
2. `dlgnsrl04/carconnect` 저장소 접속
3. `Add file` → `Upload files`
4. 이 폴더 안의 파일 4개를 모두 업로드
   - `index.html`
   - `vehicles.js`
   - `404.html`
   - `README.md`
5. 아래 초록색 `Commit changes` 클릭
6. 저장소 상단 `Settings` → 왼쪽 `Pages`
7. `Build and deployment` 항목에서
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
8. `Save` 클릭

## 고객용 주소

설정 완료 후 1~3분 뒤 아래 주소로 접속됩니다.

https://dlgnsrl04.github.io/carconnect/

## 수정/업데이트

새 차량 엑셀을 받으면 다시 홈페이지 파일을 생성해서 `index.html`, `vehicles.js`를 덮어쓰면 됩니다.

## 중요

구글드라이브 차량 사진 폴더는 고객이 볼 수 있도록 `링크가 있는 모든 사용자 보기` 권한으로 설정되어 있어야 합니다.
